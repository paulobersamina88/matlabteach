# Professor-Style MATLAB + Stiffness Method PRO

A Streamlit teaching app for introducing MATLAB and stiffness matrix formulation in a more interactive way.

## Features
- Learn tab for theory and matrix concepts
- Solve tab for live stiffness analysis
- Visualize tab for displacement and reaction plots
- MATLAB Code tab for auto-generated script
- Quiz Mode for recitation and review

## Run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```

## Recommended use
Use this in class while discussing:
1. Element stiffness matrix
2. Global matrix assembly
3. Boundary conditions
4. Displacement solution
5. Reaction recovery

## Notes
This version uses a 1D axial bar system to teach the stiffness method clearly.
